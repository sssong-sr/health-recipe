## ⚠️ 常见问题排查

### 问题 1：页面仍然空白

**可能原因：**
1. 样式文件未保存
2. 小程序未重新编译
3. 浏览器缓存

**解决方案：**
```
1. 确认 index.wxss 已保存
2. 点击编译按钮重新编译
3. 清空浏览器缓存（Ctrl+Shift+Delete）
4. 重新进入首页
```

### 问题 2：控制台显示错误

**错误示例 1：** `userGet is not a function`
- 原因：云函数未部署
- 解决：右键 `cloudfunctions/userGet` → 上传部署

**错误示例 2：** `env not found`
- 原因：环境 ID 不正确
- 解决：检查 `app.js` 中的 `env` 是否为 `cloud1-7gnwzpwhf7d728bb`

**错误示例 3：** `collection not found`
- 原因：数据库 `users` 集合不存在
- 解决：在云开发控制台创建 `users` 集合

### 问题 3：表单填写后保存失败

**可能原因：**
1. `userCreate` 云函数未部署
2. 数据库权限不足
3. 网络连接问题

**解决方案：**
```
1. 右键 cloudfunctions/userCreate → 上传部署
2. 检查云开发控制台 → 数据库 → users 集合权限
3. 检查网络连接
4. 查看控制台错误日志
```
